## 测试数据生成（脚本）

**依赖：** Python 3.x、`xlwt`

```powershell
pip install xlwt
cd D:\数据恢复Ritridata\scripts
python gen_test_office.py
```


| 路径                         | 说明                |
| -------------------------- | ----------------- |
| `G:\recovery-test\xls\`    | 脚本生成 8 个 `.xls`   |
| `G:\recovery-test\ppt\`    | 脚本下载/复制 `.ppt`    |
| `G:\recovery-test\assets\` | 可选图片，脚本复制         |
| `scripts\ppt_templates\`   | 离线时放置本地 `.ppt` 模板 |


**COM_004（内部）：** 执行 `gen_test_office.py`，确认 xls/ppt 目录齐全。
